系统理由 · Reason / Explanation

与 PDF 中 Identify / Attribute 等一致
Identify Inviter
System needs to know who sent the invitation to attribute links and issue rewards.
Attribute Return
Links inviter account, invitation link, invited friend, and return behavior.
Issue Rewards
Confirms reward ownership, prevents repeated invites, ensures rewards go to correct accounts.
Prevent Cheating
Acts as a basic threshold to limit anonymous abuse, fake returns, and repeated rewards.
Synchronize Progress
Binds personal progress (invites, rewards, leaderboard) to a real account.
